void Main\+Window\+::\+Read\+Configuration (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Reads the application configuration from settings. 

Reads and parses configuration settings from a file.

This function reads the configuration file specified by \textquotesingle{}watermeters.\+conf\textquotesingle{} located in the application\textquotesingle{}s directory. It parses key-\/value pairs separated by \textquotesingle{}=\textquotesingle{} and terminates each entry with \textquotesingle{}\texorpdfstring{$>$}{>}\textquotesingle{}. It then validates the \doxylink{class_m_d5}{MD5} checksum formed by concatenating \textquotesingle{}company\textquotesingle{} and \textquotesingle{}maximum\textquotesingle{} fields against a stored control checksum (\textquotesingle{}control\textquotesingle{}). If validation succeeds, default configuration values are updated.

If the configuration file cannot be opened or if validation fails, default configuration values are set using \doxylink{class_main_window_a20951f546fc8cf0101652bd5b55aff1b}{Set\+Default\+Configuration()}. e.\+g. company=Compania de Apa Braila\texorpdfstring{$>$}{>} archive=C\+:/\+Stand/\+Fise\texorpdfstring{$>$}{>} maximum=20\texorpdfstring{$>$}{>} certificate=CE 06.\+02-\/355/15\texorpdfstring{$>$}{>} density\+\_\+20=998\texorpdfstring{$>$}{>} control=f1807e24ccba79a76baa08194b7fa9bf\texorpdfstring{$>$}{>}

company + density\+\_\+20 =\texorpdfstring{$>$}{>} \doxylink{class_m_d5}{MD5} Here is the call graph for this function\+:
