\doxysubsection{Detailed Description}
The \doxylink{class_m_d5}{MD5} class computes \doxylink{class_m_d5}{MD5} hashes of strings or byte arrays. 

Implementation of the \doxylink{class_m_d5}{MD5} Message-\/\+Digest Algorithm.

It is not designed for speed or security, but rather as a simple implementation for educational purposes.

Usage\+: 1) Feed it blocks of uchars with \doxylink{class_m_d5_ac5ddf6cd8f940422396d321ea90ed045}{update()} 2) Finalize() 3) Get \doxylink{class_m_d5_aaf466f683b4bd8b1b66544f48bf09608}{hexdigest()} string or use \doxylink{class_m_d5}{MD5(std\+::string)}.\doxylink{class_m_d5_aaf466f683b4bd8b1b66544f48bf09608}{hexdigest()}

Assumes that char is 8-\/bit and int is 32-\/bit.

This class provides methods to calculate \doxylink{class_m_d5}{MD5} hashes for strings or byte arrays. It is based on the reference implementation of RFC 1321 by RSA Data Security, Inc.

This implementation is not intended for speed or security.
